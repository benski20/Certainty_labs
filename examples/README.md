# Certainty Labs — Examples

## Quickstart

Follow the [API docs](https://certaintylabs.ai) quickstart:

1. Create an API key in **Platform → API Keys**
2. Set your API key:
   ```bash
   export CERTAINTY_API_KEY="ck_..."
   ```
3. Install and run:
   ```bash
   pip install certaintylabs
   python examples/quickstart.py
   ```

The script exercises: health, train (built-in + from file), rerank, score, and pipeline — exactly as documented.
